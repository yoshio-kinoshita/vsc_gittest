# 目標設定

## SMARTな目標設定とは
* specific(より具体的な)
* measurable(評価可能な)
* achievable(実現可能な)
* realistic(現実的な)
* time bound(期限内に達成可能な)

## スマートゴール
明確かつ、パワルフルな目標。「きちんと準備すれば達成可能」な目標であること。
### 例
* 教養をつけるために週に２冊本を読む
* 2017年までにTOEFLで100点を取得するために英単語帳を１日20ページ進める
* 7年後の独立に向けてMBAを2020年までに取得する。そのために必要な試験勉強を１ヶ月に30ページずつ行う

# 自己PR

これまで様々な業種のWebシステムをフロントエンドからバックエンドまで開発してきました。特にコード品質とパフォーマンスにはこだわりを持っています。
決済代行プロジェクトではOSSを組み合わせ、コードレビューとコード分析を徹底。エンジニアが意識しなくても品質を確保できる環境を作りました。また、フラッシュサイトやショッピングサイトの急増に伴い、プログラム・SQLのパフォーマンスチューニングを実施。その際、デグレートしないため、テストコードを徹底しました。
常に改善を心掛け、エンジニアが楽しく正しい開発ができる環境を作るとともに、自分の身に着けた知識を活かし、世の中に貢献できればと思っております。
直近では、リソースの無い人事チームが気軽に導入できるタレントマネジメントを提供したいと考えております。
